OBIDEV FAQ

Tout ce que vous devez savoir sur nos plugins

Qu'est-ce qu'Endstone ?

+

Endstone est une API de serveur pour Minecraft Bedrock Edition permettant d'utiliser des plugins similaires à ceux de Bukkit/Spigot pour Java Edition. Nos plugins sont 100% compatibles.

Pourquoi vos plugins sont gratuits ?

+

Nous croyons en l'open-source. Nos plugins sont gratuits sous licence MIT. Vous pouvez les utiliser, modifier et redistribuer librement. Notre but : aider la communauté Endstone à grandir.

Comment installer un plugin ?

+

1. Téléchargez le fichier .pyz depuis notre GitHub
2. Placez-le dans plugins/ de votre serveur Endstone
3. Redémarrez le serveur
4. C'est tout !

Où signaler un bug ?

+

Créez une issue sur notre GitHub ou rejoignez notre serveur Discord. Nous répondons généralement sous 48h.
