FAQ

Tout ce que vous devez savoir sur nous

Pourquoi utilisé Endstone ?

+

Endstone est nouveau, optimisé et conçu pour les serveurs Bedrock modernes. Les plugins en Python/C++ ont une rapidité de développement et de maintenance à ne pas négliger. Endstone tourne sur des BDS officiels Mojang, ce qui permet des performances natives exceptionnelles. Core C++ donc 2-3 fois plus rapide que les alternatives PHP. Voici pourquoi nous avons choisi Endstone.

Pourquoi nos plugins sont gratuits ?

+

Tous nos plugins sont 100% gratuits et open-source (licence MIT) Endstone est nouveau, on construit l'écosystème ensemble pour que la communauté teste, améliore et propose des idées pour ensuite devenir une référence des plugins Endstone.

Comment installer un plugin ObiDev ?

+

1. Téléchargez le fichier .Whl depuis notre GitHub
2. Placez-le dans plugins/ sur votre serveur Endstone
3. Redémarrez le serveur
4. C'est tout !

Où signaler un bug ?

nous avons un mail support@obidev-studios.fr exactement pour ce type de demande. Nous répondons généralement sous 48h.
